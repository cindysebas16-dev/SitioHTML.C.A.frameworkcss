EVOLUCIÓN DEL HTML: de HTML 1.0 a HTML5

Línea de tiempo

HTML 1.0 (1991)

  • Primera versión creada por Tim Berners-Lee.
  • Muy básico: texto, enlaces y pocas imágenes.
  • No había diseño ni estilos.

HTML 4.01 (1999)

  • Separación de contenido y estilo (llega CSS).
  • Prioridad a la accesibilidad.
  • Deprecación de etiquetas como <font>.
  • DOCTYPE más estricto.

HTML 2.0 (1995)

  • Primera versión estandarizada.
  • Mejor manejo de errores.
  • Formularios básicos.
  • Soporte para tablas simples.

XHTML 1.0 (2000)

  • Versión estricta basada en XML.
  • Uso obligatorio de minúsculas y cierre de etiquetas.
  • No tuvo la adopción esperada.

HTML 3.2 (1997)

  • Soporte para scripts (JavaScript).
  • Tablas mejoradas.
  • Etiquetas hoy obsoletas como <font> y <center>.
  • Mezclaba estructura y diseño.

HTML5 (2014 – actualidad)

  • Estándar moderno.
  • Soporte nativo para multimedia.
  • Formularios avanzados.
  • Almacenamiento local.
  • Nuevas etiquetas semánticas:
  1. <header>
  2. <section>
  3. <article>
  4. <aside>
  5. <footer>
  6. <nav>
