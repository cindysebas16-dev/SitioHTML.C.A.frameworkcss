¿Qué es HTML?

Definición

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, listas, imágenes o tablas de datos.

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura del contenido. Mediante sus etiquetas se puede organizar y presentar la información de una forma adecuada.
