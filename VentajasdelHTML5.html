EVOLUCIÓN DEL HTML: de HTML 1.0 a HTML5

Nuevas etiquetas semánticas

Mejoran la estructura y la accesibilidad del sitio web.

Ejemplo: 
<header>
    <nav>Menú</nav>
</header>

<section>
    <article>Contenido principal</article>
</section>

<footer>© 2026</footer>

Soporte multimedia nativo

<video src="video.mp4" controls></video>

<audio src="audio.mp3" controls></audio>

<canvas> para gráficos y juegos

<canvas id="miCanvas" width="300" height="200"></canvas>

Formularios avanzados

Nuevos tipos de entrada:
  • email
  • date
  • range
  • color
  • tel
Ejemplo:

<input type="email" required>

APIs modernas

  • Geolocalización
  • Web Storage (localStorage y sessionStorage)
  • Drag & Drop
  • Aplicaciones web offline
