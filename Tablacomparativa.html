Tabla comparativa: Etiquetas obsoletas vs modernas

Etiqueta obsoleta Razón Etiqueta moderna
<font> Mezcla contenido y estilo CSS (font-size, color)
<center> No es semántica CSS (text-align)
<big> / <small> Estilo, no estructura CSS (font-size)
<u> Se confundía con enlaces CSS (text-decoration)
<applet> Insegura y obsoleta <object> o JavaScript
<frame> / <frameset> Mala accesibilidad <iframe>
<dir> Lista anticuada <ul>
<acronym> Unificada con <abbr> <abbr>
<strike> No semántica <del> o CSS
